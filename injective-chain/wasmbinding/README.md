# CosmWasm support

This package contains CosmWasm integration points.

This package provides first class support for:

- Queries
  - Auction
  - Authz
  - Exchange
    - Markets
    - Positions
  - Fee Grant
  - Insurance
  - Ocr
  - Oracle
    - Oracle Prices
  - Peggy
  - Token Factory
    - Denoms
  - Wasmx
- Messages / Execution
  - Auction
  - Exchange
  - Fee Grant
  - Insurance
  - Ocr
  - Oracle
  - Peggy
  - Token Factory
  - Wasmx

## Command line interface (CLI)

- Commands

```sh
  injectived tx wasm -h
```

- Query

```sh
  injectived query wasm -h
```
